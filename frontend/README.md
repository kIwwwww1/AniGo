# Yumivo Frontend

Фронтенд приложение для платформы Yumivo, созданное с использованием React и Vite.

## Установка

1. Перейдите в папку фронтенда:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
```

## Запуск в режиме разработки

Убедитесь, что вы находитесь в папке `frontend`, затем:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
```

## Переменные окружения

Создайте файл `.env` в корне папки `frontend`:

```
VITE_API_URL=http://localhost:8000
```

## Структура проекта

- `src/pages/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/services/` - API сервисы для взаимодействия с бэкендом
- `src/index.css` - Глобальные стили

## Интеграция с бэкендом

Фронтенд настроен на работу с FastAPI бэкендом через прокси в `vite.config.js`. 
Все запросы к `/api/*` автоматически перенаправляются на `http://localhost:8000`.

## Stagewise - Редактирование интерфейса в браузере

Stagewise интегрирован в проект и позволяет редактировать интерфейс прямо в браузере.

### Использование

1. Убедитесь, что вы находитесь в папке `frontend`:
   ```bash
   cd frontend
   ```

2. Убедитесь, что сервер разработки запущен:
   ```bash
   npm run dev
   ```

3. В отдельном терминале перейдите в папку `frontend` и запустите Stagewise:
   ```bash
   cd frontend
   npm run stagewise
   ```

После запуска вы сможете редактировать интерфейс приложения прямо в браузере, выбирая элементы и внося изменения с помощью подсказок.
